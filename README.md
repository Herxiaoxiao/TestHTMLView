# TestHTMLView
通过重写TextView的方式实现自动解析Html文本的自定义组件HtmlView，该组件主要实现的功能是解析&lt;img/>标签显示图片并添加图片的点击事件，另外在代码中也示范了一种解析自定义标签的方法（此方法参考stackoverflow上某个问题解答）；
